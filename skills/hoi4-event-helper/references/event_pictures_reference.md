## Custom Event Pictures

### Creating Custom Pictures

1. Create DDS image (recommended: 512x512)
2. Place in `gfx/event_pictures/`
3. Define in a `.gfx` file:

```
spriteTypes = {
    spriteType = {
        name = "GFX_report_event_my_custom_picture"
        texturefile = "gfx/event_pictures/my_custom_picture.dds"
    }
}
```

4. Use in events:
```
country_event = {
    picture = GFX_report_event_my_custom_picture
}
```

---

## Tips for Picture Selection

1. **Match Theme**: Choose pictures that match your event's theme
   - War events → battle/combat pictures
   - Diplomacy → negotiations/handshake pictures
   - Economy → factory/production pictures

2. **Country-Specific**: Use country-specific pictures for national events
   - German events → German soldiers/tanks
   - Soviet events → Soviet army
   - etc.

3. **News Events**: Use dramatic pictures for news events
   - `GFX_news_event_war_declared` for war declarations
   - Generic news pictures for less dramatic events

4. **Fallback**: When in doubt, use generic pictures
   - `GFX_report_event_generic` works for almost anything
   - Safe choice that won't look out of place

5. **Consistency**: Use similar pictures for related events
   - Event chains should use thematically similar pictures
   - Country-specific chains should use that country's pictures

## Troubleshooting

### Picture Not Showing
1. Check spelling of GFX reference
2. Verify picture exists in game files
3. For custom pictures, check .gfx definition
4. Ensure DDS file is in correct location

### Picture Shows as Placeholder
1. Picture may be DLC-specific
2. GFX name might be wrong
3. File might be corrupted

### Best Practice
Use generic pictures (`GFX_report_event_generic_*`) for maximum compatibility.
